Layout Demo

This page demonstrates the full letterbox layout: fixed header with user card, dark sidebar, scrollable main content, and fixed footer.

Page Structure
<div class="page-wrapper">
  <header class="site-header site-header-dark">...</header>
  <div class="page-body">
    <aside class="site-sidebar sidebar-left">...</aside>
    <main class="main-content">...</main>
  </div>
  <footer class="site-footer site-footer-dark">...</footer>
</div>
Features
  • Letterbox layout — header and footer are fixed, main content scrolls independently
  • Stacking sidebars — add multiple .site-sidebar elements inside .page-body
  • Responsive collapse — sidebars auto-hide below 768px, toggle with .show
  • Dark variants — .site-header-dark, .site-sidebar-dark, .site-footer-dark
  • User card — hover popup with avatar, name, role, details, actions
