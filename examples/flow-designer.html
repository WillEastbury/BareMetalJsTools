BareMetal.FlowCanvas

A component-based, drag-and-drop nested flow designer. Every atomic statement is a box; block statements (If, For each, On event) are boxes that contain boxes, so code lives visually inside loops and choices. Drag from the palette onto the canvas, grab a box by its ⠇ grip to move or re-nest it, and edit fields inline. The step model is identical to BareMetal.WorkflowPico, so it compiles straight to PicoScript and runs on the same VM.
